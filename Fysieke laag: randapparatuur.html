de fysieke laag van randapparatuur is de laag in een computernetwerk die verantwoordelijk is voor de verbindingen tussen apparaten. Deze laag zorgt voor hoe gegevens worden overgedragen via bijvoorbeeld: kabels, radiogolven. Voor bijvoorbeeld toetsenborden wordt de informatie door gegeven via kabels die aangesloten aan het toetsenbord. De processor (CPU) is het brein van een computer die alle instructes uitvoert & ook berekeningen maakt om taken te doen. Het is een van de belangrijkste componenten van een computer. Bij elke nieuwe telefoon die uitkomt wordt er ook een nieuwere & betere CPU in de telefoon gedaan om de telefoon sneller & beter te maken. Kloksnelheid is de snelheid waarmee de CPU zijn taken doet. Het wordt gemeten in gigahertz (GHz). hoe nieuwer de CPU hoe hogere kloksnelheid de CPU heeft. Mijn eigen CPU in mijn pc heeft een 4.1 GHz kloksnelheid. verwerkingssnelheid is de snelheid waarmee een computer of CPU gegevens kan verwerken en de taken kan uitvoeren.
